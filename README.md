## Getting Started

Check This [Issue](https://github.com/nsccpccoe/nscc-backend/issues/1) to Track Progress

Prerequisites

```bash
npm install -g firebase-tools
```

Login with firebase

```bash
firebase login
```

Start Emulator

```bash
firebase emulators:start
```

Build Watch Cloud Functions

```bash
cd functions
npm run build:watch
```
